Morse-Protocol
=============
Data Transmission over Morse Code

Requirements
------------
- [portaudio](http://www.portaudio.com/)
- ffi-portaudio Gem
- base32 Gem

Try it!
-------
- *Source*: `ruby sample/send.rb`
- *Recpient*: `ruby sample/recieve.rb`
