Morse-Protocol
=============
Data Transmission over Morse Code

Requirements
------------
- [portaudio](http://www.portaudio.com/)
- ffi-portaudio Gem
- base32 Gem

Try it!
-------
- *Recpient*: `ruby sample/listen.rb`
- *Source*: `ruby lib/morse_protocol.rb`
